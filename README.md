# React Native Job Search App

This is a React Native application that allows users to search for jobs posted on various sites, including LinkedIn, Glassdoor, and Indeed. The app has search functionality and the ability to filter full-time, part-time, and contractor positions. The app also includes two lists: popular jobs and nearby jobs. The popular jobs list scrolls horizontally, while the nearby jobs list scrolls vertically.

## Features
- Search for jobs on LinkedIn, Glassdoor, and Indeed
- Filter job search results by full-time, part-time, and contractor positions
- Two lists: popular jobs and nearby jobs
- Click on a job to view job details, including qualifications, job description, and responsibilities
- Click a button at the bottom of the job details page to be taken to the site where the job was posted

## Future Improvements
- Set up a profile to store the user's data, such as CV
- Save liked jobs
- Implement login functionality

## Getting Started
To get started with this app, simply clone this repository and run the following commands:

- npm install
- npm start

## API Access
This app accesses the JSearch API to retrieve job search results. You will need to obtain an API key to use this app. Please follow the instructions on the JSearch website to obtain an API key.

## Dependencies
- React Native
- JSearch API
